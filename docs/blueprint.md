# **App Name**: Dominio Gramatical

## Core Features:

- Contenido de Gramática Interactiva: Presenta explicaciones detalladas, reglas, fórmulas, ejemplos y trucos mnemotécnicos en español para los 5 temas gramaticales, incluyendo tablas de conjugación y listados específicos como adverbios de frecuencia y participios irregulares. Toda la interfaz y el texto explicativo están en español (Latinoamérica).
- Minipruebas de Comprensión: Cada sección gramatical incluye un cuestionario de 3 a 5 preguntas de opción múltiple, proporcionando retroalimentación instantánea (correcto/incorrecto con explicación) y un seguimiento de la puntuación que se guarda localmente.
- Seguimiento de Progreso Local: Permite al usuario marcar secciones como estudiadas y guarda su progreso junto con los resultados de las pruebas en 'localStorage'. Incluye una barra de progreso global visible en la parte superior.
- Navegación Dinámica de Secciones: Implementa una navegación pegajosa (barra lateral en escritorio, superior en móvil) que permite saltar directamente a cualquier sección, resaltando visualmente la sección activa al desplazarse.
- Herramienta de Expansión de Ejemplos con IA: Una herramienta que, para una regla gramatical seleccionada, puede generar oraciones de ejemplo adicionales en inglés, con sus traducciones al español y explicaciones de cómo se aplica la regla en el nuevo contexto.
- Reiniciar Progreso: Un botón de fácil acceso para restablecer completamente el progreso del usuario guardado localmente, con un paso de confirmación.
- Diseño Responsivo Completo: Asegura que la aplicación sea totalmente adaptable y funcional en diversos dispositivos, desde escritorios hasta teléfonos móviles, manteniendo una experiencia de usuario fluida.

## Style Guidelines:

- Esquema de color oscuro: un fondo casi negro con un tinte cálido sutil (#1D1A16), complementado por texto en blanco hueso cálido (#D9D7D4) para facilitar la lectura en entornos oscuros.
- Color primario: un ocre/dorado intenso y cálido (#E4B067) utilizado para resaltar información clave, botones de acción (CTAs) y acentos generales que evoquen sofisticación y claridad.
- Color acento: un rojo quemado profundo y vibrante (#B82214) que proporciona contraste para elementos secundarios, avisos o para destacar la corrección en ejemplos.
- Tarjetas de ejemplos correctos: utilizan un color verde bosque apagado (#3E723D) como fondo o acento, transmitiendo un tono de aprobación.
- Tarjetas de ejemplos incorrectos o errores: emplean un rojo óxido oscuro (#54221D), similar al color acento pero más tenue, para indicar advertencia o error.
- Fuente para títulos de sección: 'Playfair', una serif moderna, que aporta una sensación editorial elegante y premium.
- Fuente para cuerpo de texto y UI: 'Inter', una sans-serif grotesca que ofrece legibilidad clara y un estilo objetivo y moderno para el contenido principal y elementos de la interfaz.
- Fuente monoespacio: 'Source Code Pro' para la visualización de fórmulas gramaticales y fragmentos de código, asegurando alineación y claridad técnica.
- Íconos minimalistas y estilizados en un diseño de línea o glifo, que sean legibles sobre el fondo oscuro y complementen la estética editorial sofisticada.
- Un diseño adaptable que cambia de una vista de dos columnas (barra lateral de navegación y contenido principal) en escritorio a una sola columna optimizada para dispositivos móviles.
- Tarjetas de contenido para organizar las explicaciones, ejemplos y cuestionarios, con distintivos visuales (como distintivos numerados y etiquetas de color) para cada sección.
- Visualización de fórmulas gramaticales en un estilo de píldora/insignia oscura con tipografía monoespacio, para destacarlas claramente dentro del contenido.
- Animaciones sutiles y suaves en el desplazamiento (scroll) y la navegación entre secciones, mejorando la experiencia del usuario sin distraer del contenido didáctico.
